# Sistema de Pedidos – Java

Projeto desenvolvido em Java com o objetivo de praticar conceitos de Programação Orientada a Objetos.

## Descrição
Sistema simples de gestão de pedidos, contendo as entidades Cliente, Produto e Pedido, além de um enum para controle de status do pedido.

O projeto foi desenvolvido como exercício acadêmico, com foco em:
- Estruturação de classes
- Organização em pacotes
- Uso de enum
- Boas práticas de POO

## Estrutura do Projeto
Organização dos pacotes:
- entities
- enums
- repositories
- controllers

A classe Main é responsável pela execução do sistema.

## Status do Projeto
Em desenvolvimento.

Funcionalidades de exportação de pedidos em TXT e XML estão em fase de implementação.
